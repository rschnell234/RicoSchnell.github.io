Morse Code App

This is an App developed by me and two other students in a University module called Mobile App Development. Its main emphasis lied on developing an App with Flutter and Dart. But we also learned several things about UI/UX and about managing user data in a database. We used Firebase as a way to manage our users. Firebase was very helpful, because it was very easy to implement and connect to Flutter. The app that we implemented ended up being an app where the user can communicate with his friends via morse. It was aimed at users that wanted to learn morse code. My main working areas on this project were the handling of the user-system. and the tutorial.
